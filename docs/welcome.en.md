# Welcome !

<div class="doc-hero">

![icon](icons/welcome.png)

<div> 

This documentation is intended to describe and demonstrate the capabilities of the Create Aeronautics mod blocks when used with the CC: Tweaked mod via the Lua programming language. <br><br> The main goal of the project is to create a user-friendly reference guide for interacting with mechanisms, peripherals, and automation systems.

</div>
</div>

---

What's included in this documentation:
- descriptions of methods and APIs;
- examples of Lua scripts;
- interactions between blocks;
- practical considerations for real-world projects;
- non-standard use cases;
- known limitations and bugs.

> 💡 Note:
> This documentation is suitable for both beginners learning automation in Minecraft and experienced developers of complex control systems, transportation systems, and computing complexes.
