# Поворотный подшипник

<div class="doc-hero">

![icon](icons/swivel_bearing.png)

<div>

Данный блок используется в моде для поворота частей конструкций на определённый угол, а также позволяет отдельным частям конструкции свободно вращаться (элемент может свободно «болтаться» на нём).

</div>
</div>

| Режим работы               | Описание                                                                                                          |
| -------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| Всегда заблокирован        | Поворот конструкции жёстко зависит от вращения шестерни. Не реагирует на сигналы редстоуна                        |
| Не полностью заблокирован  | Если подать сигнал редстоуна на блок, конструкция становится свободно вращающейся, иначе остаётся заблокированной |
| Не полностью разблокирован | Если подать сигнал редстоуна на блок, конструкция блокируется, иначе свободно вращается                           |
| Всегда разблокирован       | Конструкция на блоке всегда свободно вращается. Не реагирует на сигнал редстоуна                                  |

> 💡 Примечание  
> Через функции CC Tweaked невозможно менять режим работы блока. Можно только подавать на него сигнал редстоуна, если установлен соответствующий режим.

---

## Функции, предоставляемые блоком

Подключить блок к терминалу из мода CC Tweaked можно так:

```lua
local swivel_bearing = peripheral.wrap("swivel_bearing_0") -- имя можно получить при активации в чате
```
Так это выглядит в мире (скрин для тестов):![imgs](imgs/swivel_bearing_0.png) 



---

## функции что нам доступны

| функция             | результат      | описание                                          |
| ------------------- | -------------- | ------------------------------------------------- |
| getTargetAngle()    | число (0–360°) | Установленный угол поворота подшипника в градусах |
| getTargetAngleRad() | число (0-2π)   | Установленный угол поворота подшипника в радианах |

---

## Пример использования в коде

```lua
-- подключаем блок к терминалу
local swivel_bearing = peripheral.wrap("swivel_bearing_0")

-- допустим, задан угол 45 градусов

-- получаем угол в градусах
local angle_in_degrees = swivel_bearing.getTargetAngle()

-- получаем угол в радианах
local angle_in_radians = swivel_bearing.getTargetAngleRad()

```
