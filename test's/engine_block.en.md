# White Lamp

![icon](icons/base_icon.png)

Decorative industrial light source.

Perfect for futuristic interiors, laboratories and machine rooms.

---

## Usage In Game

The block emits strong white light.

Can be controlled using Lua peripherals.

---

## Lua Example

```lua
local lamp = peripheral.wrap("left")

lamp.turnOn()

print("Lamp enabled")
```

---

## Notes

Supports automation systems and redstone integration.
